# Customer Relationship Management System (CRM)

## Project Description
This project is a web-based Customer Relationship Management (CRM) system designed to help businesses manage customer data efficiently. The system allows users to store, search, update, and manage customer information in a centralized platform.

## Features
- Create customer profiles  
- View customer records  
- Search customer data  
- Update customer information  
- Delete customer records  
- Manage customer purchase history  
- Manage loyalty information  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- GitHub  

## Development Approach
This project is developed using the Kanban methodology. Jira is used to create tasks, assign responsibilities, and track progress. GitHub is used for version control and collaboration, while Clockify is used for time tracking.

## Purpose
The purpose of this project is to apply software engineering concepts such as planning, system design, teamwork, and version control in a real-world scenario.

## Project Evolution
This project was developed after evaluating and improving an earlier idea (Student Task Manager). This helped us better understand the importance of choosing a meaningful problem and designing a system with real-world value.
