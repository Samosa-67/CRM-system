# Customer Relationship Management System (CRM)

## Project Description

This project is a web-based Customer Relationship Management (CRM) system developed to improve how customer information is stored, accessed, and managed. The system centralizes customer-related data into a single platform and allows users to manage customer profiles efficiently through a responsive dashboard interface.

The CRM system includes customer management functionality, dashboard analytics, revenue tracking, customer tier classification, and local storage integration. The project was developed using HTML, CSS, and JavaScript while following software engineering practices such as Jira Kanban workflow, GitHub version control, Clockify time tracking, and Essence Kernel evaluation.

## Development Approach

The project was developed using Agile and Kanban methodologies. Jira was used for task management and workflow tracking, while GitHub branches and pull requests were used for version control and collaborative development.

Clockify was used for PSP-based time tracking to monitor task progress and improve productivity throughout the project lifecycle.
